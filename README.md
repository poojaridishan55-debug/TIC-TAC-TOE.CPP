# TIC-TAC-TOE.CPP
Hello everyone, I made a simple tic-tac-toe game in C++ using some concepts which I learned in my fyjc(class 12th). I used these concepts:- Game loop using "while(condition)", Conditional logic using "if" and "if-else", Class and objects (Object Oriented programming -OOP), Multidimensional arrays. I used Cxxdroid app on android to create this.
